Features

Constellapedia is on active development. This are some of the features implemented and to be implemented in the near future.

Powerful visualizations

Constellapedia leverage the power of VisJS to render hundreds of nodes on screen without lag


progress: 100%

Crowdsourced

Constellations are created and maintained by the community. Soon Constellapedia will add user permissions and levels to manage editing of current constellations


progress: 50%

OTSM-TRIZ based framework

Constellapedia is based on an improved version of OTSM-TRIZ. With lots of community feedback, we will fine tune the framework even more


progress: N/D

Graph database

Because of how Constellapedia maps problems, the usage of graph database technologies makes exploring large datasets lighting fast


progress: 90%

Crowdfundable nodes

Nodes can be publicly fund with FIAT (USD/EUR). In the future with the usage of crypto token might be enforced (ERC20 and/or others)


progress: 20%

Research tools

We are working on the making of data science tools to run graph algorithms against the data sets, for relation discovery and analysis of different perspectives.


progress: 5%
